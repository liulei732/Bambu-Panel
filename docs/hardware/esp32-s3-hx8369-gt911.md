# ESP32-S3 4.3 寸 HX8369 + GT911 硬件 Profile

来源参考：`/Users/liulei/esp32/LVGL_4.3inch_GT911/docs/BOARD_ADAPTATION.md`

## 硬件摘要

| 项目 | 值 |
| --- | --- |
| MCU | ESP32-S3 |
| LCD 驱动 | HX8369 |
| LCD 接口 | Intel 8080 / I80，8-bit 并口 |
| 分辨率 | 800 x 480 |
| 像素格式 | RGB565 |
| 触摸 IC | GT911 |
| 触摸接口 | I2C |
| 稳定 LCD PCLK | 30 MHz |

## LCD 引脚

| 信号 | GPIO |
| --- | --- |
| D0 | 46 |
| D1 | 3 |
| D2 | 8 |
| D3 | 18 |
| D4 | 17 |
| D5 | 16 |
| D6 | 15 |
| D7 | 7 |
| WR / PCLK | 10 |
| CS | 12 |
| DC | 11 |
| RST | 9 |
| BL | 6 |

显示方向使用：

```c
esp_lcd_panel_swap_xy(panel_handle, true);
esp_lcd_panel_mirror(panel_handle, true, false);
```

## GT911 触摸

| 信号 | GPIO |
| --- | --- |
| SDA | 39 |
| SCL | 38 |
| RST | 4 |

I2C 频率：`400000 Hz`

当前板子刷机实测 GT911 地址：`0x5d`。`0x14` 保留为 fallback address，兼容同款控制器在不同上电/复位配置下的地址差异。

坐标变换：

```text
swap_xy = true
invert_x = false
invert_y = true
```
